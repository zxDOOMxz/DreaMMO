**Механика:**

- Уведомление по прибытию
- Автообновление списка зон при прибытии

#### 4. Боевая система (Lineage 2 style)

**Боевой лог:**
- ⚔️ КРИТИЧЕСКИЙ УДАР (красная граница)
- ❌ Промах (серая граница)
- ☠️ Убийство (зелёная граница)
- 💥 Обычный удар (синяя граница)
- Автопрокрутка, хранит 50 последних записей

**Панель характеристик:**
- 💪 Урон: min-max
- 🎯 Шанс попадания: % (от ловкости)
- ⚡ Шанс крита: % (от удачи)
- 🛡️ Шанс блока: % (от ловкости + выносливости)
- ⏱️ Скорость атаки: атак/мин (от ловкости)
- 🛡️ Броня: значение (от выносливости)

**Формулы:**
```
Урон = base_damage + (Сила - 10) / 2
Крит = базовый 5% + (Удача - 10) * 0.3%
Попадание = базовый 85% + разница ловкости * 0.5%
Блок = базовый 10% + (Ловкость-10)*0.2% + (Выносливость-10)*0.1%
Скорость атаки = 60 * (1 + (Ловкость-10)*0.05)
```

**Система опыта (Lineage 2):**
- Моб = уровню или выше: 100% опыта
- Моб на 3-5 lvl ниже: 75% опыта, 90% золота
- Моб на 6-10 lvl ниже: 50% опыта, 75% золота
- Моб 11+ lvl ниже: 0% опыта, 50% золота (только лут)

#### 5. Система групп

**Панель группы:**
- 👥 Создание группы (название)
- 👑 Показывает лидера
- 📊 Участников: текущие/максимум (6)
- 💚 HP участников в реальном времени
- 🚪 Кнопки: "Распустить группу" / "Покинуть группу"

**Механика:**
- Макс. 6 участников
- Общий опыт за убийства
- Квесты засчитываются всем участникам группы
- Лидер может приглашать игроков
